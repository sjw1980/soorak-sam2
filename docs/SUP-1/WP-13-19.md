### 3. 결과물

#### 3.1 부적합 (Non-conformities)

- NCF-01 (Major): 언어/플랫폼 불일치 — `PROJECT-KICKOFF`의 `프로그래밍 언어: Python` 및 품질 기준 일부(예: `메모리: RAM 64KB`)는 현재 문서·환경과 상충하거나 현실적이지 않음.

  - 근거: [PROJECT-KICKOFF](../PROJECT-KICKOFF.md) 4. 기술 플랫폼, 6. 품질 기준
  - 영향: 비기능 요구사항·테스트 기준 불일치로 인한 시험 설계 오류 가능

- NCF-02 (Major): 품질 기준의 일부 수치 불합리 — `메모리: RAM 64KB` 등은 Python 기반 개발 환경에 부적합(임베디드 C/C++ 기준으로 보임).

  - 근거: [PROJECT-KICKOFF](../PROJECT-KICKOFF.md) 6. 품질 기준
  - 영향: 테스트·커버리지 목표의 실현성 저해

- NCF-03 (Minor): `MAN3-PP-0001` 프로젝트 계획서의 `작성자` 필드 미기입 — 추적성·책임 소재 불분명

  - 근거: [MAN3-PP-0001](../MAN-3/MAN3-PP-0001-project-plan.md) 상단 메타 정보

- NCF-04 (Minor): SRS(WP-17-11)의 베이스라인 미확정 — Phase 1 종료 조건(요구사항 베이스라인 승인) 충족 전제 미완료

  - 근거: MAN3-PP-0001 Phase 1 산출물 목록

- NCF-05 (Minor): 요구사항↔테스트 추적성 매트릭스(WP-13-22) 미비 — 추적성 증거 미흡
  - 근거: 프로젝트 내 traceability 시각화는 있으나 정형화된 WP-13-22 기록 미확인
