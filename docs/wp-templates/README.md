# WP 양식 목록 (Work Product Templates)

> ASPICE v3.1 기반 작업 산출물 양식 인덱스

---

## 사용 방법

1. 산출물 작성이 필요한 경우, 아래 목록에서 해당 WP ID의 양식 파일을 복사하여 `docs/{프로세스}/` 디렉터리에 저장
2. 양식의 `[기술할 내용]` 플레이스홀더를 실제 내용으로 대체
3. 문서 정보 (버전, 날짜, 상태, 작성자) 갱신
4. 변경 이력 테이블 갱신
