# 프로젝트 킥오프 — 사람이 결정해야 할 항목

> **용도**: AI Agent 기반 ASPICE 프로젝트 시작 전, 사람이 반드시 결정·입력해야 하는 항목입니다.
> 아래 항목을 작성한 뒤 Agent에게 전달하면, 이후 프로세스(MAN.3 → SWE → SPL.2)가 자동으로 진행됩니다.
>
> **버전**: v1.0 / 2026-04-19
> **상태**: Draft

## 1. 제품 비전 (Product Vision)

| 항목               | 내용                                                                                                |
| ------------------ | --------------------------------------------------------------------------------------------------- |
| 제품명             | CLI Calculator                                                                                      |
| 제품 개요          | 사칙 연산 및 기본 계산 기능을 제공하는 CLI 기반 계산기, 입력은 실수 및 Root 입력 방법을 제공해야 함 |

## 4. 초기 요구사항 (Initial Requirements)

### 4.1 핵심 기능 요구사항

> 제품이 반드시 수행해야 하는 기능을 나열합니다.

1. 사용자 입력을 파싱하여 사칙연산 수행
2. 0으로 나눗셈 시 에러 메시지 반환
3. 제곱근 및 지수 입력 지원

### 4.2 비기능 요구사항

| 항목   | 기준                      |
| ------ | ------------------------- |
| 성능   | 응답시간 100ms 이내       |
| 메모리 | RAM 64KB 이하             |
| 보안   | 입력 검증 필수            |
| 이식성 | Linux/Windows 크로스 빌드 |

### 4.3 제외 사항 (Out of Scope)

- 제약 없음
